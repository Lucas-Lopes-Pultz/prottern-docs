## describe

The **describe** command shows information about the template passed as a parameter.


```command
$ prottern describe <template-name>
```

output example:

```
>> name
   | project-web

>> type
   | Local

>> owner
   | user1

>> created at
   | 2021-07-10 14:02:55.657383900 UTC

>> paths
   | index.html
   | src
   | src\script.js
   | src\style.css.css
```
